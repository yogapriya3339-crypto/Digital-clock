# ⏰ Digital Clock Web Application

A simple and beginner-friendly digital clock built using **HTML, CSS, and JavaScript**.

## 📌 Project Description

This project displays the current time in hours, minutes, and seconds and updates automatically every second using JavaScript.

## 🚀 Features

- Displays current time in real-time.
- Updates every second automatically.
- Simple and clean user interface.
- Beginner-friendly project for learning web development.

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```text
digital-clock/

├── w2.html

├── script1.js

```

## ▶️ How to Run

1. Download or clone the repository.
2. Open the project folder in Visual Studio Code.
3. Open `w2.html`.
4. Run the file using Live Server or open it directly in your browser.
5. The digital clock will appear and update every second.

## 📖 Concepts Used

- HTML Structure
- CSS Styling
- JavaScript Functions
- Date Object
- DOM Manipulation
- `setInterval()` Function

## 📸 Output

- Displays a heading **Digital Clock**.
- Shows the current time in `HH:MM:SS` format.
- Updates the displayed time every second.
